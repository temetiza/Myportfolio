Project Description

The goal of this project is to build replica websites for four major tech brands - Apple, Zoom, Netflix and Amazon. These websites will replicate the core functionality and key pages of the original sites. The replica sites is responsive and optimized for desktop, tablet and mobile viewing. They utilize HTML, CSS and JavaScript for frontend development. A content management system like WordPress is used to facilitate content updates.

The replica sites have placeholder/dummy content and basic e-commerce functionality where applicable (like add to cart). Advanced features like payments integration or video playback has been included. The goal is to gain experience in replicating major websites with responsive design, consistent branding and key functionality. Usability testing may also be conducted to evaluate how closely the replica sites match the usability of the originals.

In addtion I developed the login and registration pages for evagadi website to allow customers to securely create accounts and access their order history. The goal was to provide a smooth user experience for common authentication tasks.
